#     SDLC-and-STLC

This explains where QA fits into development.

Requirements
     ↓
Design
     ↓
Development
     ↓
Testing
     ↓
Deployment
     ↓
Maintenance


Important concept:
Modern QA starts early—not after development is finished.STLC (Software Testing Life Cycle)

This is the QA-specific workflow:
Requirement Analysis
        ↓
Test Planning
        ↓
Test Case Creation
        ↓
Environment Setup
        ↓
Test Execution
        ↓
Defect Reporting
        ↓
Test Closure
